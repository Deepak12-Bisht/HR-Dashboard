# 📊 HR Analytics Dashboard

An interactive HR Analytics Dashboard built to provide actionable insights into workforce data, helping organizations make data-driven decisions related to employee performance, retention, and recruitment.

# 🚀 Overview

This dashboard visualizes key HR metrics such as employee demographics, attrition trends, recruitment sources, and performance indicators. It enables HR teams to quickly identify patterns and improve workforce management strategies.

# ✨ Features
# Employee Overview
Detailed employee profiles (name, role, salary, manager, performance score)
Personal attributes (gender, marital status, DOB, joining date)

# Attrition Analysis
Termination reasons breakdown
Voluntary vs involuntary termination comparison

# Demographics Insights
Gender distribution
Marital status analysis
Race diversity visualization

# Recruitment Analytics
Hiring source effectiveness (Indeed, LinkedIn, Google Search, etc.)
Comparative hiring trends

# Interactive Filtering
Department-level filtering
Dynamic employee selection

# 🛠️ Tech Stack
Data Visualization: (Power BI)
Data Processing: (Excel)

# Key takeaway:
The dashboard reveals that the majority of employee attrition (~85–90%) is voluntary, primarily driven by dissatisfaction, better compensation opportunities, and career changes. The workforce is relatively balanced in terms of gender (56.6% male, 43.4% female) but shows limited diversity, with around 60% of employees being White. A significant portion of employees (44%) are single, indicating an early-career demographic. Recruitment efforts are most effective through external platforms like Indeed (87 hires) and LinkedIn (76 hires), while channels such as the company website (13 hires) and CareerBuilder (23 hires) underperform. Overall, the insights highlight key areas for improvement in employee retention, compensation strategy, and diversity hiring.
